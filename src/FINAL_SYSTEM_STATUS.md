## 🎓 **USAGE GUIDE:**

### **Getting Started:**
1. Open application in browser
2. Login with Super Admin credentials
3. Create users for different roles
4. Assign permissions
5. Set up inventory
6. Add parties (customers/suppliers)
7. Start billing!

### **Super Admin Credentials (Default):**
- **Email:** superadmin@servespares.com
- **Password:** super123

### **First-Time Setup:**
1. Login as Super Admin
2. Go to Settings → Configure company details
3. Go to User Roles → Create users for your team
4. Go to Parties → Add your suppliers and customers
5. Go to Total Inventory → Add your products
6. Start using the system!
